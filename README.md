# WorldShop
Proyecto Final de Universidad 
Trabajando un Ecommerce con los siguientes lenguajes:
En el ambiente Front End: Scraping (puppeteer) web, Node.js, JavaScript nativo, EJS, BootStrap, etiquetas HTML y estilos SCSS-CSS.
En el ambiente Back End: Mysql.

La Finalidad del proyecto es crear un Ecommerce con el modelo de trabajo DropShipping, el cual consisten en vender sin poseer un stock propio, proporcionando productos
de diferentes partes del mundo a través del Scraping web proporcionado por la página.
